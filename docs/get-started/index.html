about:blanker

A standard of disguising, the about:blanker opens a new about:blank window with a iframe containing the website. Simple to create, simple to use, and usually quite effective. Requires access to bookmarklets, otherwise this will not work.

This method is easy to do - just drag the link below to your bookmarks bar. Then, click the new bookmark!

A common disguise method, save a file on your Chromebook, then open it. Just an HTML file with an iframe. Some schools block this - if it's blocked, try Save as .htm.
